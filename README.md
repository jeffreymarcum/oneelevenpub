# One Eleven Publishing

React website scaffold built with Vite.

## Setup

1. Install dependencies:
	npm install
2. Start development server:
	npm run dev
3. Build for production:
	npm run build

## Logo Placement

Place your provided logo image at:

public/logo.png

The homepage is already wired to use this file. If the image is missing, the site shows a simple fallback mark.
